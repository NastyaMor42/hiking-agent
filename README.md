# 🥾 Hiking Agent – מתכנן טיולים חכם בישראל

אפליקציה חכמה שמציעה מסלולי טיול בישראל לפי העדפות המשתמש, בשילוב AI וחיפוש בזמן אמת.

---

## ✨ מה האפליקציה עושה

* 🔍 מחפשת מסלולי טיול אמיתיים באינטרנט
* 🧠 משתמשת ב-AI כדי לבחור ולהתאים המלצות
* 🌍 תומכת בעברית מלאה (RTL)
* 📱 מאפשרת שליחת מסלול לוואטסאפ
* ⏰ (בהמשך) תזכורות אוטומטיות + מזג אוויר

---

## 🧱 מבנה הפרויקט

```
app.py         # UI (Streamlit)
agent.py       # לוגיקת הסוכן
search.py      # חיפוש (SerpAPI)
whatsapp.py    # שליחת הודעות
weather.py     # מזג אוויר
```

---

## 🚀 איך מריצים

1. מתקינים ספריות:

```
pip install -r requirements.txt
```

2. מגדירים מפתחות (Streamlit secrets):

```
OPENAI_API_KEY=...
SERPAPI_KEY=...
TWILIO_SID=...
TWILIO_TOKEN=...
```

3. מריצים:

```
streamlit run app.py
```
